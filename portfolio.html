PROJECT_02Desktop App

Vizyon Almanya Veritabanı Yönetimi

Vizyon Almanya projesi kapsamında Microsoft SQL Server (MSSQL) ve C# kullanılarak geliştirilmiş masaüstü tabanlı bir veritabanı yönetim uygulaması. Tablo oluşturma, veri ekleme/güncelleme/silme, tablolar arası ilişki kurma ve dinamik SQL sorguları gibi kapsamlı CRUD işlemlerini içermektedir.
